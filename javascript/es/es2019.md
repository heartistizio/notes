# ES2019

## Array.prototype.{flat,flatMap}

- `flat(depthLevel = 1)` - returns flattened array, ie `[[1,2,3], 4]` will be `[1, 2, 3, 4]`, takes optional depth level

- `flatMap(callback)` - a map operation with flattening afterwards

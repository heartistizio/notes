# ESlint

## Plugins

### eslint-plugin-jsx-a11y

Accessibility rules in JSX.

### eslint-plugin-react

Lets eslint understand react.

## Version

`"detect"` will allow eslint to detect React's version.

```json
  "settings": {
    "react": {
      "version": "detect"
  }
```  

### eslint-plugin-react-hooks

ESlint rules for hooks from React team.
